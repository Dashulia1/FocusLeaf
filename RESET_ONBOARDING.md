# Как сбросить онбординг для тестирования

Если вы хотите снова увидеть экран онбординга при запуске приложения, используйте один из следующих способов:

## Способ 1: Через настройки Android (Рекомендуется)

1. Откройте **Настройки** на вашем устройстве
2. Перейдите в **Приложения** (Apps) или **Управление приложениями**
3. Найдите приложение **FocusLeaf**
4. Нажмите на него
5. Нажмите **Хранилище** (Storage)
6. Нажмите **Очистить данные** (Clear Data) или **Очистить кэш и данные**
7. Подтвердите действие
8. Запустите приложение заново — онбординг должен появиться

## Способ 2: Через ADB (для разработчиков)

Если у вас подключен Android Debug Bridge, выполните команду:

```bash
adb shell pm clear com.dasasergeeva.focusleaf2
```

Это очистит все данные приложения, включая флаг онбординга.

## Способ 3: Переустановка приложения

1. Удалите приложение с устройства
2. Установите его заново
3. Онбординг появится при первом запуске

## Способ 4: Программный сброс (для разработки)

В коде приложения можно временно добавить сброс онбординга. Метод `PreferencesManager.resetOnboarding()` уже доступен.

---

**Примечание:** После сброса онбординга вам также нужно будет пройти авторизацию заново, если вы вышли из аккаунта.
