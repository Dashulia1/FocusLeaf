# Инструкция по добавлению изображений для онбординга

Для отображения ваших изображений в онбординге, добавьте файлы в папку:
```
app/src/main/res/drawable/
```

## Названия файлов (в порядке экранов онбординга):

1. **`onboarding_0_character.png`** (или `.jpg`, `.webp`)
   - Экран 0: Монстр-персонаж (розово-фиолетовый монстр на черном фоне)
   - Размеры: рекомендуется 800x800px или больше

2. **`onboarding_1_organization.png`**
   - Экран 1: Человек с телефоном (организация контента)
   - Размеры: рекомендуется 800x800px или больше

3. **`onboarding_2_analytics.png`**
   - Экран 2: Телефон с аналитикой и графиками
   - Размеры: рекомендуется 800x800px или больше

4. **`onboarding_3_success.png`**
   - Экран 3: Женщина с галочкой успеха
   - Размеры: рекомендуется 800x800px или больше

## Формат файлов:
- Поддерживаемые форматы: `.png`, `.jpg`, `.webp`
- Рекомендуется использовать `.png` для прозрачности или `.webp` для лучшего сжатия

## После добавления:
1. Поместите файлы в папку `app/src/main/res/drawable/`
2. Переименуйте их согласно списку выше
3. Пересоберите проект (Build → Rebuild Project)
4. Запустите приложение

## Примечание:
Layout файлы уже обновлены для использования этих изображений. Просто добавьте файлы с правильными именами в папку drawable.
